Door Monitoring with Reed Switch - IoT project

This project is a door monitoring system using an ESP8266, a reed switch, LEDs, and a buzzer. It can detect when the door is open or closed and send notifications to a Telegram bot. The system also features WiFi setup with WiFiManager for easy network configuration.
